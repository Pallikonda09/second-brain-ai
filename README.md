# Second Brain AI

Second Brain AI is an AI-powered knowledge management system that allows users to capture, organize, and retrieve information efficiently.

## Live Demo
https://second-brain-ai-liard-five.vercel.app

---

## Features

- Create and manage notes, links, and insights  
- AI-powered summarization  
- Automatic tag generation  
- Search and filter knowledge items  
- Conversational query system  
- Public API for external access  

---

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS  
- Backend: Next.js API Routes  
- Database: MongoDB Atlas  
- AI: Groq API (LLaMA 3.3 70B)  
- Deployment: Vercel  

---

## AI Features

### Summarization
Generates concise summaries for stored knowledge.

### Auto Tagging
Assigns relevant tags based on content.

### Conversational Query
Allows users to query their knowledge base using natural language.
